C++ heap: priority_queue

  2020-05-02  Comments

假如需要使用到 heap, C++ 裡的 heap 可使用 <queue> 裡面的 priority_queue。

值得注意的是, priority_queue 預設是一個 max heap,想要 min heap 的話,要定義自己的 comparator。

下面用 Leetcode 中的一題 merge K sorted lists, 來說明怎麼使用 priority_queue。

Merge K sorted lists

題目連結

這個題目我們可以用 priority_queue 來解：

ListNode* mergeKLists(vector<ListNode*>& lists) {
  ListNode dummy;
  dummy.next = NULL;
  ListNode* node = &dummy;

  auto comp =
    [](const ListNode* e1, const ListNode* e2)
    { return e1->val > e2->val; };

  priority_queue<ListNode*, vector<ListNode*>, decltype(comp)> q(comp);

  for(auto node : lists) {
    if (node != NULL) {
      q.push(node);
    }
  }

  while(q.size() > 0) {
    ListNode* top = q.top();
    q.pop();
    node->next = top;
    node = node->next;
    if (top->next != NULL) {
      q.push(top->next);
    }
  }

  return dummy.next;
}

可以注意到我們定義了一個 comp, 利用這樣的方式把 priority_queue 變成 min heap。

priority_queue in STL

深入看一下 priority_queue 的宣告。

template<
  class T,
  class Container = std::vector<T>,
  class Compare = std::less<typename Container::value_type>
  > class priority_queue;

priority_queue() : priority_queue(Compare(), Container()) { }
explicit priority_queue(const Compare& compare)
: priority_queue(compare, Container()) { }
explicit priority_queue( const Compare& compare = Compare(),
                         const Container& cont = Container() );

所以我們需要給一個 Compare 。這邊 lambda function 做出一個 comparator，再利用 decltype 得到這個 lambda function 的 type。 decltype 真是蠻好用的，適合這種 type signature 非常複雜的情況。
